# Ronin City - Living Blockchain Visualizer

Real-time pixel art visualization of Ronin blockchain activity.

## Features

- City mode with pixel buildings
- Highway mode with scrolling block buses
- 12 ecosystem projects tracked
- Live RPC data ingestion and rendering
